# Batman Portfolio

This repository has been converted into a **React** project using **Vite**. The main entry page is `index.html` which mounts React into a `#root` div. The primary application component lives in `src/App.jsx` and is bootstrapped by `src/main.jsx`.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

To build for production:

```bash
npm run build
npm run preview
```

Tailwind CSS is configured and used throughout the project; see `tailwind.config.cjs` and `postcss.config.cjs` for settings.
